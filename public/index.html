Memory Management: Weak and Unowned

May 16, 2019
Automatic Reference Counting Swift uses Automatic Reference Counting (ARC) to track and manage your app’s memory usage. You can check Swift docs for more detail information to memory management. Whenever you assign a class instance to a property, constant, or variable, that property, constant, or variable makes a strong reference to the instance. The reference is called a “strong” reference because it keeps a firm hold on that instance, and does not allow it to be deallocated for as long as that strong reference remains. …
